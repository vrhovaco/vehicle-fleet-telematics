# ORBIT FLEET – Vehicle Telematics Platform

Premium frontend-only vehicle fleet telematics dashboard with a 3D vehicle viewer, live telemetry simulation, route replay, alerts, and interactive analytics.

## Live Demo

https://vrhovaco.github.io/vehicle-fleet-telematics/

---

## Features

- 3D vehicle viewer with rotation and camera controls
- Live telemetry simulation
- Vehicle switching
- Route replay and projected path
- Operational alert feed
- Performance insights
- Responsive premium dashboard UI
- Frontend-only architecture, ready for GitHub Pages

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Three.js
- OrbitControls
